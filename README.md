Analyse des performances commerciales
Contexte

Dans le cadre de ce projet, une entreprise souhaite mieux comprendre son activité commerciale à partir de ses données clients, commandes et produits.

L'objectif est d'identifier les principaux leviers de performance, d'analyser les comportements d'achat et de mettre en évidence des opportunités d'amélioration grâce à une approche basée sur les données.

Objectifs de l'analyse

Cette étude vise à répondre aux questions suivantes :

Quelles sont les villes les plus performantes en termes de chiffre d'affaires ?
Quels produits et catégories génèrent le plus de ventes ?
Quels sont les clients les plus actifs et les plus rentables ?
Comment évoluent les ventes selon les périodes observées ?
Quels indicateurs permettent de mesurer la performance commerciale globale ?
Jeux de données utilisés

Le projet repose sur trois sources de données :

Customers : informations clients et localisation.
Orders : historique des commandes et quantités achetées.
Products : catalogue produits et catégories associées.

Recommandations

À l'issue de cette analyse, plusieurs axes d'amélioration peuvent être envisagés :

Renforcer les actions commerciales dans les villes les plus performantes.
